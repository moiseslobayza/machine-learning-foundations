# Fundamentos de Machine Learning

Colección de notebooks prácticos desarrollados durante mi formación en Ciencia de Datos e Inteligencia Artificial.

Este repositorio documenta mi progreso en los fundamentos del Machine Learning clásico, abarcando preprocesamiento de datos, aprendizaje supervisado y aprendizaje no supervisado.

## Contenidos

| Notebook | Tema |
|---|---|
| `01_feature_scaling.ipynb` | Escalamiento de variables y preprocesamiento |
| `02_clustering.ipynb` | Aprendizaje no supervisado y clustering |
| `03_regularization.ipynb` | Regularización de modelos |
| `04_linear_regression.ipynb` | Regresión lineal |
| `05_classification.ipynb` | Modelos de clasificación y evaluación |

## Flujo de Machine Learning

```text
Preprocesamiento de datos
        ↓
Transformación de variables
        ↓
Entrenamiento del modelo
        ↓
Predicción
        ↓
Evaluación del modelo
```

## Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Objetivo del repositorio

El objetivo de este repositorio es documentar la implementación práctica de conceptos fundamentales de Machine Learning utilizando Python y Scikit-learn.

Los notebooks se enfocan en comprender el proceso de modelado, las técnicas de preprocesamiento, el comportamiento de los modelos y su evaluación.
